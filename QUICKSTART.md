# Быстрый старт

## Шаг 1: Получите токен бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен вида `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

## Шаг 2: Узнайте свой Telegram ID

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Бот отправит вам ваш ID (например, `123456789`)

## Шаг 3: Настройте переменные окружения

Скопируйте `env.example` в `.env` и заполните:

```bash
cp env.example .env
```

Отредактируйте `.env`:
```
BOT_TOKEN=ваш_токен_от_botfather
ADMIN_ID=ваш_telegram_id
```

## Шаг 4: Запустите бота

### Вариант 1: Через скрипт (рекомендуется)

```bash
./run.sh
```

### Вариант 2: Напрямую

```bash
go run .
```

### Вариант 3: Скомпилировать и запустить

```bash
go build
BOT_TOKEN="ваш_токен" ADMIN_ID="ваш_id" ./whitelist
```

## Готово!

Бот запущен! Найдите его в Telegram по имени, которое вы задали в BotFather, и отправьте команду `/start`

## Основные команды

**Для пользователей:**
- `/start` - Начать работу
- `/apply` - Подать заявку
- `/status` - Проверить статус

**Для админа:**
- `/pending` - Посмотреть ожидающие заявки
- Кнопки в сообщениях для одобрения/отклонения

## Поддержка

При возникновении проблем проверьте:
- Правильность токена бота
- Правильность ADMIN_ID
- Наличие интернет-соединения
- Логи в консоли
